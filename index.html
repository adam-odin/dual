These are the steps I had to go through to set up dual booting with linux ubuntu on my Windows laptop:

  1. Use Disk Manager to set aside storage space on your laptop for ubuntu (ideally 30GB+)
  2. Use Rufus to convert your linux image file download into a USB drive
  3. Change Windows restart to Safe Mode then Change your UEFI/BIOS settings to AHCI from RAID On
  4. Use your USB drive to install ubuntu when you reboot
  5. Thinking about which packages to install, keep or get rid of
  6. Dreaded Warnings encountered along the way and how you might fix them!
  7. Troubleshooting and using logs files to optimise ubuntu set up, including graphics
