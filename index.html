How to dual boot Linux Ubuntu on a Windows laptop

                

                

                
                

                    

                        

                            
The Odin Project

                            
One of the first requirements of the Odin Project is to install linux.
 I hope this guide helps you dive in to dual booting even more confidently than I did.

                            
These are the essential steps:

                            
    
                                
  1. Use Disk Manager to set aside storage space for ubuntu (ideally 30GB+)
    2. 
                                
  2. Use Rufus to convert your linux image file download into a USB drive
    3. 
                                
  3. Change Windows restart to Safe Mode before changing UEFI/BIOS setting SATA Operation to "AHCI" from "RAID On"
    4. 
                                
  4. Use your USB drive to install ubuntu
    5. 
                            

                        
                            
And these are some follow up considerations, especially if you're tight for space.

                            I set aside 19GB on my SSD and 200GB on my HDD which seems to be working well.
